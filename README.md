# Human Substrate

The economy is a torus. Labor flows in. Capital accumulates at the core. The surface closes back on itself, and what looks like growth is mostly circulation — the same value moving faster through a tightening loop.

AI doesn't break the loop. It accelerates it. The question isn't whether automation displaces workers. The question is whether the substrate — human attention, judgment, creativity, the thing the whole system runs on — gets any return on what it generates.

This repo is the working record. Observable model first, then every iteration of the Economic Torus visualization, then the writing. Public and timestamped. The thinking, not just the finished object.
